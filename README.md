<h1 align="left">Wireless Profile Dump</h1>

###

<p align="left">A Python tool for viewing saved Wi‑Fi profiles and recovering your own network keys using Windows CMD. Built for educational and administrative purposes.</p>

###

<h2 align="left">About project</h2>

###

<p align="left">A lightweight Python script that extracts and displays saved Wi‑Fi profiles and their security keys from Windows using CMD commands. Intended for system administrators and users who need to recover forgotten Wi‑Fi passwords on their own devices.</p>
